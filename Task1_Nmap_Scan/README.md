# Task 1: Basic Network Scanning using Nmap

## Internship
AICTE OIB-SIP Cybersecurity Internship

## Objective

To perform a basic network scan to identify open ports, services, and possible security risks using Nmap.

## Tool Used

- Nmap
- Zenmap (Graphical Interface)

## Scan Target

Localhost (127.0.0.1)

## Scan Type

Intense Scan

## Findings

The scan identified active ports and services running on the system.

Open ports may increase the attack surface of a system if not properly secured.

## Security Recommendations

- Disable unnecessary services
- Keep software updated
- Monitor open ports regularly
- Use firewall protection

## Conclusion

This task helped in understanding network reconnaissance and identifying exposed services using Nmap.
