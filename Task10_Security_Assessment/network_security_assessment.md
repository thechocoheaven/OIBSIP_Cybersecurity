# Network Security Assessment Report

## Objective

To analyze network security using scanning and packet analysis techniques.

## Tools Used

- Nmap
- Wireshark

## Methodology

1. Performed network scanning using Nmap.
2. Identified open ports and services.
3. Captured and analyzed network traffic using Wireshark.

## Findings

### Open Services

Open ports may increase attack surface and should be monitored.

### Network Traffic

Captured packets showed normal ICMP and DNS communication.

## Risks

- Unnecessary open ports
- Information exposure
- Weak network monitoring

## Recommendations

- Close unused ports
- Enable firewall protection
- Regular vulnerability assessment
- Monitor network traffic

## Conclusion

Security assessment helps identify weaknesses and improve overall network security.
