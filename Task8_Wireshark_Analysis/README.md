# Task 8: Network Traffic Capture using Wireshark

## Internship

AICTE OIB-SIP Cybersecurity Internship

## Objective

To capture and analyze network traffic using Wireshark.

## Tool Used

Wireshark

## Process

1. Started packet capture on active network interface.
2. Generated network traffic using ping command.
3. Applied ICMP filter to analyze packets.
4. Saved captured traffic for analysis.

## Analysis

Captured packets included ICMP communication generated by ping requests.

Wireshark helps security analysts monitor network activity and identify suspicious traffic.

## Security Importance

Packet analysis helps in:
- Detecting abnormal traffic
- Troubleshooting networks
- Identifying possible attacks

## Conclusion

This task provided practical understanding of network packet capture and analysis.
