# AICTE OIB-SIP Cybersecurity Internship

This repository contains the tasks completed during the **AICTE OIB-SIP Cybersecurity Internship**.

The internship focused on practical cybersecurity concepts including network scanning, vulnerability assessment, security analysis, and cybersecurity research.

---

## Completed Tasks

## Task 1: Basic Network Scanning using Nmap

**Objective:**
Performed network scanning to identify open ports and available services.

**Tools Used:**

* Nmap
* Zenmap

**Deliverables:**

* Nmap scan results
* Documentation of findings
